### 题解
斐波那契数列可以采用递归解法，但是递归法重复计算部分太多，因此选择使用动态规划解答。
动态规划的特点是：最优子结构、无后效性、子问题重叠。
```javascript
function Fibonacci(n)
{
    let [a,b] = [0,1];
    while(n--){
        b=a+b;
        a=b-a;
    }
    return a;
}
module.exports = {
    Fibonacci : Fibonacci
};
```
