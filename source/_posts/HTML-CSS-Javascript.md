# 一、HTML
HTML(Hypertext Markup Language)超文本标记语言。

## 1.标签
标签一般成对出现，但是存在一些自结束标签（单标签），如：
```html
<img>、<img/>、<input>、<input/>
```
## 2.注释
注释不能嵌套
```html
<!-- 注释内容 -->
```
## 3.标签的属性
在标签（开始标签或自结束标签）还可以设置属性
