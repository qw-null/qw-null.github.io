### 题解
树的深度 = max(左子树深度，右子树深度)+1
```javascript
function TreeDepth(pRoot)
{
    //递归出口
    if(pRoot===null) return 0;
    let left = TreeDepth(pRoot.left);
    let right = TreeDepth(pRoot.right);
    return 1 + (left>right?left:right);
}

```
