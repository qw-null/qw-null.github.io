### 题解
交换二叉树的左右子树
```javascript
function Mirror( pRoot ) {
    //递归出口
    if(pRoot === null){
       return pRoot;
       } else {
        [pRoot.left,pRoot.right] = [pRoot.right,pRoot.left]
    }
    Mirror(pRoot.left);
    Mirror(pRoot.right);
    return pRoot;
}
```

### 补充知识
二叉树的答题模板
```javascript
function traverse(TreeNode root){
    // 前序遍历
    traverse(root.left)
    // 中序遍历
    traverse(root.right)
    // 后序遍历
}
```
递归的中心思想：
+ 考虑出口 （ if 部分）
+ 处理当前节点
+ 递归处理下个节点

二叉树问题的递归出口，一般包括：
```javascript
// 1.节点为空（到达叶子节点）
pRoot === null
```
