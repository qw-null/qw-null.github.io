Linux环境下配置git
2024-09-24
字数总计: 293|阅读时长: 1 分钟

总共分为两个步骤：安装 + 配置，最终实现通过Git拉取代码。

1.安装Git

得到返回信息 “git version XX.XX.XX.XX” 表明Git已经安装成功。

2.配置Git

分为两个步骤：初始化Git并生成授权证书 + 代码仓库（github）配置

2.1 初始化Git

2.2 生成授权证书

执行该命令后，会有3次确认：
第一次：生成公钥和私钥的文件名称(默认即可)
第二次：生成证书的密码(默认为空即可)
第三次：确认证书密码(默认为空即可)

2.3查看生成公钥

执行ls命令后，会看到两个文件id_rsa.pub,id_rsa，其中id_rsa.pub就是生成的公钥。

2.4 配置代码仓库

登录github，在用户设置中，找到SSH and GPG keys，点击New SSH key，将生成的公钥复制到Key中，然后点击Add SSH key即可。


运行命令ssh git@github.com, 出现以下信息则表示配置成功。
